GDClick, version 1.014, made by watakak.
https://github.com/watakak/GDClick

To setup, create a folder on your pc and drop all files from .zip archive to folder.

To convert: 
1. Put your GDH macro/record into main folder.
2. Open program.
3. Type name of .txt macro. (only name)
4. Set FPS of macro.
5. Press enter, wait a little, and you done!
